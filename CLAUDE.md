# Git Commit Rules
- 커밋 메시지에 Co-Authored-By 줄을 절대 포함하지 않는다
- 커밋 메시지에 Claude, AI, LLM 관련 언급을 하지 않는다

# 학습 대화 규칙
- 대화하며 지식을 확장하고, 학습 노트도 함께 업데이트한다
- ~요 형태로 친근하게 대화하고, 노트도 같은 톤으로 작성한다

# 노트 작성 형식
- 수식: LaTeX (`$...$`, `$$...$$`)
- 흐름도, 구조도: Mermaid (` ```mermaid `)
- 좌표 그래프, 도형: matplotlib 이미지 생성 후 첨부
- 행렬, 벡터 표현: LaTeX `\begin{bmatrix}` 사용
- 노트 언어: 한국어 (영어 용어 함께 표기)
